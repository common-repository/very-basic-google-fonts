== Description ==

The simple, bare-bones way to include Google Fonts to the wp_head(). It isn’t fancy but it gets the job done.

== Frequently Asked Questions ==

= What’s the plugin like? =

As simple as it can be. Just a sub-page of Settings with a textarea.

= How do I choose which fonts I want? =

Go to Settings/Google Fonts and enter the font names in the box, one per line.

E.g.
Open Sans
Vollkorn
